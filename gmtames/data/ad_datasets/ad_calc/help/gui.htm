Data structure

Data must be organised as samples (molecules) as rows and variables (molecular descriptors) as columns. Training and test set must be organised in two different data matrices. Type:

load example_data

on the MATLAB command window to see an example of data structure.

[-> top]


Starting the graphical interface

The Applicability Domain toolbox can work both on the MATLAB command window (see details) or with its graphical interface. The graphical interface enables you to do all the steps of the analysis. In order to start the graphical interface, type the following code in the MATLAB command window:

ad_gui

the main form of the graphical interface will appear:

In order to activate the menu of the form, both training and test datasets must be loaded. In the window, two listboxes will show the details of the loaded data (on the left) and of the loaded (or calculated) AD methods (on the right).

[-> top]


Loading training and test set

Training and test dataset can be loaded directly from the matlab workspace or from a matlab data file, using the file menu. Here it is also possible to save and export AD results to excel files.

[-> top]


How to calculate AD methods

In order to calculate AD methods select the calculate menu. This will be enables only if both training and test sets have been loaded in the toolbox.

[-> top]


Viewing results

After the AD calculation, the model window in the main form is updated with the AD details (list of AD methods calculated, number of test samples excluded from the model AD). Then, it is possible to see further details in the results menu.
