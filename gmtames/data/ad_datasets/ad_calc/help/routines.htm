This is a list of the main routines you can use to calculate AD by means of the Applicability Domain toolbox for MATLAB:

ad_gui

ad_gui opens a GUI for calculating all AD methods provided in the toolbox; in order to open the graphical interface, just type on the matlab command line:

ad_gui

there are no inputs; data can be loaded and saved directly from the graphical interface.

[-> top]


ad_model

ad_model can be used to calculate all AD methods directly on the MATLAB command window, without the graphical interface, just type on the matlab command line:

res = ad_model(Xtrain,Xtest,options,whattodo);

type "help ad_model" on the MATLAB command window for further informations on the input, options and output of this function.
