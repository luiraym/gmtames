Test set results

Once the AD methods have been calculated, the model window in the main form is updated with the AD details (list of AD methods calculated, number of test samples excluded from the model AD). It is possible the see the a resume of the results achieved on the test set by clicking "results->test set results". The following informations for each calculated AD method will be shown: options used to calculate the AD method, number of test samples inside the AD, number of test samples outside the AD, list of test samples outside the AD. This can be exported as excel file too.

[-> top]


Consensus

A consensus analysis of all AD methods can be analysed by selecting "results->view consensus". A figure will appear with frequencies of test samples placed out of the AD by the calculated methods.

[-> top]


William plot

By selecting "results->view William plot", a new form will appear. This analysis is available only if the AD method based on leverage is calculated. In this new form, it is possible to load the experimental and calculated responses for both training and test sets. Then, a plot of leverages versus standardised residuals is shown. Warning thresholds for both leverages and residuals are marked in red.

Results related to the k optimisation

If an automatic (or user) selection of the best k value on the basis of a montecarlo procedure has been used for the kNN with variable k method (see details in this section), then results of the k optimisation can be analysed by clicking "results->view k results". A box-and-whisker plot of distribution of test samples (%) retained within the AD for different k values during k-optimization is shown. Details on this plot are given in the following paper: F. Sahigara, D. Ballabio, R. Todeschini, V. Consonni, Defining a novel k-Nearest Neighbours approach to assess the applicability domain of a QSAR model for reliable predictions, Journal of Cheminformatics (2013), 5:27, 1-9.
