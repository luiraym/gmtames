Loading training and test sets

Traingin and test sets must be organised in two different data matrices. Data sets can then be loaded directly from the matlab workspace or from a matlab data file. In order to load training and test data, select in the file menu "file->load training set" and "file->load test set".

Saving and exporting AD results

AD results can be saved from the file menu ("file->save ad results"). The results are saved as structure in the matlab workspace. For further information on the structure of AD results look the help provided in the ad_model matlab m file. However, consider that a binary vector defining which test samples are inside (1) or outside (0) the AD defined by means of each AD method is provided in the result structure.
Moreover, a resume of the results achieved on the test set can be exported as excel file ("file->export test results"). An excel file will be saved in the folder where the toolbox is placed, with the following informations for each calculated AD method: options used to calculate the AD method, number of test samples inside the AD, number of test samples outside the AD, list of test samples outside the AD.
